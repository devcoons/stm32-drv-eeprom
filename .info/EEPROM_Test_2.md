# Test 2

**Board** : NUCLEO-L552ZE-Q

**EEPROM** : U2 (**I2C**)


# Steps:

In the .ioc file:
- Enable **SCL** on **PB8**
- Enable **SDA**  on **PB9**
- Enable **I2C** in **fast mode ** to set the speed at 400 kHz.
